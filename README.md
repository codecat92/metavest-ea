# Metavest EA

Automated Expert Advisor (EA) for MetaTrader 4, built with MQL4.

## Overview

Metavest EA is an automated trading bot designed to run on the MetaTrader 4 platform. It executes buy and sell decisions based on predefined technical analysis strategies, with built-in risk management through Stop Loss and Take Profit mechanisms.

## Project Structure
metavest/
├── src/
│   ├── experts/       # Main EA files (.mq4)
│   ├── indicators/    # Custom indicators
│   └── libraries/     # Reusable functions
├── docs/              # Documentation and strategy notes
├── tests/             # Backtest results
└── README.md




## Tech Stack

- Language  : MQL4
- Platform  : MetaTrader 4
- Broker    : JDR Securities
